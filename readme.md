# KodiPay
A simple app using laravel and vue

## Local SetUp
### Clone Repository
```
git clone https://github.com/gathuku/kodipay.git
```
### Install dependancies
```
composer install
&
npm install
```

### Database
```
php artisan migrate --seed
```

## Technologies Used
### Front End
- Boostrap
- VueJs
- Vue Router

### Backend
- Laravel (Framework for PHP artisans)
